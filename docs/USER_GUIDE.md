## 1. Introduction

The **Innovation Idea Tracker** is a web application designed to facilitate the submission, tracking, and evaluation of innovative ideas. Users can propose their ideas, categorize them, and engage in feedback through voting and comments.

### Key Features:
- **Submit Ideas**: Share your innovative concepts with the community
- **Vote**: Support ideas you believe in with upvotes/downvotes
- **Comment**: Provide constructive feedback and suggestions
- **Track Progress**: Monitor the status of submitted ideas

## 3. User Registration

To submit ideas and participate in discussions, you must first create an account.

### Steps:
1. Click **"Get Started"** or **"Sign In"** on the homepage
2. Click **"Create one now"** link
3. Fill in the registration form:
   - **Username**: Your display name (3-80 characters)
   - **Email**: Valid email address
   - **Password**: At least 6 characters
   - **Confirm Password**: Repeat your password
4. Click **"Create Account"**
5. You will be redirected to the login page

## 5. Submitting Ideas

Once logged in, you can submit innovative ideas to the platform.

### Steps:
1. Click **"Submit Idea"** in the navigation bar
2. Fill in the idea form:
   - **Title**: A clear, concise title (5-200 characters)
   - **Category**: Select from available categories
   - **Description**: Detailed explanation (minimum 20 characters)
3. Click **"Submit Idea"**

### Tips for Great Ideas:
- Be specific about the problem you're solving
- Include potential benefits and impact
- Mention resources or tools needed
- Keep it clear and understandable

## 9. Admin Features

Administrators have additional capabilities:

### Admin Dashboard:
- View platform statistics
- Manage user roles
- Approve/reject ideas
- Manage categories

### Changing User Roles:
1. Access **Admin Dashboard**
2. Find the user in the Users table
3. Select new role from dropdown:
   - **Submitter**: Can submit, vote, comment
   - **Reviewer**: Can also change idea status
   - **Admin**: Full access

### Changing Idea Status:
1. View idea detail page or Admin Dashboard
2. Select new status from dropdown
3. Status updates immediately

## 10. Troubleshooting

### Common Issues:

**Can't register?**
- Ensure email is valid and not already registered
- Password must be at least 6 characters
- Username must be 3-80 characters

**Can't log in?**
- Verify email and password are correct
- Check for typos
- Try the password reset if available

**Can't submit idea?**
- Ensure you're logged in
- Title must be 5-200 characters
- Description must be at least 20 characters

**Vote not working?**
- Make sure you're logged in
- Check your internet connection
- Refresh the page and try again
